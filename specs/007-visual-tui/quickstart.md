# Quickstart: Visual TUI Controls

## Keyboard Controls

| Key | Action |
| --- | --- |
| **Space** | Toggle Play/Pause |
| `←` / `→` | Seek Backward/Forward |
| `↑` / `↓` | Volume Up/Down |
| `m` | Toggle Mute |
| `n` | Next Track |
| `p` | Previous Track |
| `q` or `Ctrl+C` | Quit |

## Interface

```text
[                     Music Title                     ]
[                    [ PLAYING ]                      ]

                00:45 / 03:30
         [████████░░░░░░░░░░░░░░░░░░░░]

                      Vol: 100%

[Space] Toggle  [Arrow] Seek/Vol  [M] Mute  [Q] Quit
```
