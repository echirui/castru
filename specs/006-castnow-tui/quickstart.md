# Quickstart

## Controls

| Key | Action |
|-----|--------|
| `Space` | Toggle Play/Pause |
| `Right` | Seek Forward (30s) |
| `Left` | Seek Backward (15s) |
| `Up` | Volume Up (+5%) |
| `Down` | Volume Down (-5%) |
| `m` | Toggle Mute |
| `n` | Next Track |
| `p` | Previous Track |
| `q` / `Esc` | Quit |

## CLI Output

When running `castru cast ...`, the terminal transforms into the player interface:

```text
[Playing] 01:23 / 04:50 [==============>                 ] Vol: 100%
```

Log messages (if any) will appear above the status line.
